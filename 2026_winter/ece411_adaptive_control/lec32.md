## Adaptive Regulation

* Now that we have all the pieces required, we can put them together to perform adaptive regulation
* Consider the system $\begin{aligned}[t] x(k + 1) &= Ax(k) + Bu(k) + E\xi(k) \\ \xi(k + 1) &= S\xi(k) \\ e(k) &= Cx(k) + D\xi(k) \end{aligned}$ where:
	* $S, E, D$ are unknown
	* $A, B, C$ are known
	* $e(k)$ is measurable

\noteThm{For the exosystem $$\begin{aligned}[t] \xi(k + 1) &= S\xi(k) \\ d(k) &= \Gamma\xi(k) \end{aligned}$$ where $d(k) \in \reals$, assuming $(\Gamma, S)$ is observable, $(F, G)$ controllable, $\sigma(F) \cap \sigma(S) = \emptyset$, and $F$ is Schur stable, then there exists a nonsingular $M \in \reals^{q \times q}$ which solves the Sylvester equation $MS = FM + G\Gamma$.}

* With the coordinate transformation $w(k) = M\xi(k)$, we get the exosystem $\begin{aligned}[t] w(k + 1) &= Fw(k) + Gd(k) \\ d(k) &= \psi^Tw(k) \end{aligned}$
	* Note that since $w(k + 1) = (F + G\psi^T)w(k)$, we have $\sigma(S) = \sigma(F + G\psi^T)$
* We solve the regulator equations: $\begin{aligned}[t] \Pi S &= A\Pi + B\Gamma + E \\ 0 &= C\Pi + D \end{aligned}$, for which a solution exists if:
	1. $(A, B)$ is controllable
	2. $(C, A)$ is observable
	3. $\sigma(S)$ are *simple* (i.e. algebraic multiplicity 1) and on the unit circle in $\complex$
	4. *Nonresonance condition*: $\det \mattwo{A - \lambda I}{B}{C}{0} \neq 0, \forall \lambda \in \sigma(S)$
		* We are checking if any of the frequencies in the exosystem will be cancelled by the plant
		* This is similar to making sure there are no pole zero cancellations between the exosystem and plant
* Let the error $z(k) = x(k) - \Pi\xi(k)$, which has dynamics $\begin{aligned}[t] z(k + 1) &= Az(k) + Bu(k) - B\Gamma\xi(k) \\ &= Az(k) + Bu(k) - B\psi^Tw(k) \\ e(k) &= Cz(k) \end{aligned}$
* We use the controller $u(k) = u_s(k) + u_{im}(k)$
	* $u_s(k)$ is the stabilizer, and we expect $u_s(k) \to 0$
	* $u_{im}(k)$ contains the internal model, and we expect this to be a steady state
* We use the disturbance observer $\hat z_d(k + 1) = A\hat z_d(k) + Bu(k) + L_d(e(k) - C\hat z_d(k))$ where we choose $L_d$ such that $A - L_dC$ is Schur stable
	* Let the estimation error $\tilde z_d(k) = \hat z_d(k) - z(k) \implies \tilde z_d(k + 1) = A_d\tilde z_d(k) + Bd(k)$
	* Define the filtered disturbance $d_f(k) = C\tilde z_d(k) = C\hat z_d(k) - e(k)$, which can be interpreted as $d(k)$ filtered through the LTI system $\begin{aligned}[t] \tilde z_d(k + 1) &= A_d\tilde z_d(k) + Bd(k) \\ d_f(k) &= C\tilde z_d(k) \end{aligned}$
		* Note that $d_f(k)$ is measurable
		* Note that technically, depending on the initial conditions, $d_f(k)$ will have transients, but $d(k)$ is steady-state; therefore we rely on the lemma below

\noteThm{For the exosystem $$\begin{aligned}[t] w(k + 1) &= Fw(k) + Gd(k) \\ d(k) &= \psi^Tw(k) \end{aligned}$$ Let $A_d = A - L_dC$ be Schur stable, then there exists initial conditions $z_d(0) \in \reals^n$ such that the steady-state $d_{f,ss}(k)$ obtained by $$\begin{aligned}[t] z_d(k + 1) &= A_dz_d(k) + Bd(k) \\ d_{f,ss}(k) &= Cz_d(k) \end{aligned}$$ can be generated by a model $$\begin{aligned} w_f(k + 1) &= Fw_f(k) + Gd_{f,ss}(k) \\ d_{f,ss}(k) &= \psi^Tw_f(k) \end{aligned}$$ and there exists a nonsingular $H_f \in \reals^{q \times q}$ such that $w(k) = Hw_f(k)$.}

* Intuitively, the above is due to the fact that LTI systems don't change frequency content, so the filtered disturbance $d_f(k)$ should have the same frequencies as the unfiltered $d(k)$; therefore $d_f(k)$ can be generated with a system with the same $F$ and $G$ that generated $d(k)$, and the two systems differ only by a coordinate transformation
* To get $w_f(k)$, we build the observer $\alignedeqntwo[t]{\hat w_f(k + 1)}{F\hat w_f(k) + Gd_f(k)}{F\hat w_f(k) + G(C\hat z_d(k) - e(k))}$
	* This leads to exponential convergence of $w_f(k) \to w(k)$
* Now the disturbance becomes $\alignedeqntwo[t]{d(k)}{\Gamma\xi(k)}{\psi^Tw(k)}{\psi^TH_f^{-1}w_f(k)}{\psi _f^Tw_f(k)}{\psi _f^T\hat w_f(k) + \varepsilon(k) \Ttag{where }\varepsilon(k) \to 0}$
* Finally, $\alignedeqntwo[t]{z(k + 1)}{Az(k) + Bu(k) - B\psi^Tw(k)}{Az(k) + Bu(k) - B\psi _f^Tw_f(k)}$
	* This suggests we choose $u_{im}(k) = \hat\psi _f^T(k)\hat w_f(k)$ where $\hat\psi _f$ is our parameter estimate
	* Now $\begin{aligned}[t] z(k + 1) &= Az(k) + Bu_s(k) + B\tilde\psi _f^T(k)\hat w_f(k) + \varepsilon(k) \\ e(k) &= Cz(k) \end{aligned}$
		* This is not quite the dynamic error model yet, due to the fact that $A$ might not be Schur stable, and the additional $Bu_s(k)$
		* However if $A$ is stable, then we may set $u_s(k) = 0$ and directly obtain the dynamic error model
* In case $A$ is unstable, $\alignedeqntwo[t]{d_f(k)}{H_d(z)\left[d(k)\right] \Ttag{where }H_d(z) = C(zI - A_d)^{-1}B}{H_d(z)\left[\psi^Tw(k)\right]}{H_d(z)\left[\psi _f^Tw_f(k)\right]}{\psi _f^TH_d(z)I\left[w_f(k)\right] + \varepsilon(k) \Ttag{swapping lemma}}{\psi _f^TH_d(z)I\left[\hat w_f(k)\right] + \varepsilon(k)}$
	* Define $\hat d_f(k) = \hat\psi _f^T(k)H_d(z)I\left[\hat w_f(k)\right]$, which is measurable
	* Let the augmented error $e_a(k) = \hat d_f(k) - d_f(k) = \tilde\psi _f^T\hat w_a(k)$ where $\hat w_a(k) = H_d(z)I\left[\hat w_f(k)\right]$
		* This is also measurable since $d_f(k)$ can be computed from $e(k)$
* The final control design: $\begin{aligned}[t] \hat z_d(k + 1) &= A\hat z_d(k) + Bu(k) + L_d(e(k) - C\hat z_d(k)) \\ d_f(k) &= C\hat z_d(k) - e(k) \\ \hat w_f(k + 1) &= F\hat w_f(k) + Gd_f(k) \\ u_{im}(k) &= \hat\psi _f^T(k)\hat w_f(k) \\ \hat\psi _f(k + 1) &= \hat \psi _f(k) - \gamma(k)e_a(k)w_a(k) \\ \gamma(k) &= \frac{\bar\gamma}{1 + \norm{w_a(k)}^2}, \bar\gamma \in (0, 2) \end{aligned}$
