\begin{itemize}
\tightlist
\item
  Interest: \(i_e = \left(1 + \frac{i}{m}\right)^m - 1\) for \(m\)
  subperiods per compounding period
\item
  Mortgage: \(A = P(A/P, i, N)\) where \(A\) is the monthly payment,
  \(P\) is the principal, \(i\) is the monthly interest rate

  \begin{itemize}
  \tightlist
  \item
    To recalculate monthly payment after a term:
    \(P(F/P, i, n) - A(F/P, i, n)(P/A, i, n)\), then calculate new
    payment
  \end{itemize}
\item
  Bond: \(P = C(P/A, i, N) + F(P/F, i, N)\) where \(P\) is the sell
  price, \(C\) is the coupon payment, \(F\) is the face value, \(i\) is
  the yield

  \begin{itemize}
  \tightlist
  \item
    First coupon paid 1 period later; higher yield makes bond sell for
    cheaper
  \item
    If yield is higher than coupon, \(P < F\); if yield is lower than
    coupon, \(P > F\)
  \item
    When maturity period is not aligned with coupon payments, calculate
    price in the past and apply interest forward
  \end{itemize}
\item
  CAPM: \(\mathbb E[R_i] = r_f + \beta _i(\mathbb E[R_{MP}] - r_f)\)
  where \(r_f\) is the risk-free rate,
  \(\displaystyle\beta _i = \frac{\sigma _{i,MP}}{\sigma _{MP}^2} = \frac{\rho _{i,MP}\sigma _i}{\sigma _{MP}}\)
\item
  \(R_\text{WACC} = \frac{E}{E + D}R_E + \frac{D}{E + D}R_D(1 - t)\)
\item
  If projects have different time horizons: use LCM of project lifetimes
  (repeated lives), or use annual worth
\item
  IRR: The interest (discount) rate that makes present worth 0; higher
  is more profitable, but note investment time/size

  \begin{itemize}
  \tightlist
  \item
    Simple investments have a unique IRR, non-simple ones may have
    multiple solutions
  \end{itemize}
\item
  (Discounted) payback period: time taken to recoup an investment; does
  not account for earnings after payback
\item
  Incremental analysis (mutually exclusive projects): order by
  increasing first costs (starting with do nothing); calculate
  incremental IRR between new option and current option using
  \(\Delta PW\) and \(\Delta A\); if lower than MARR, keep current
  option; otherwise switch
\item
  Depreciation:
  \(BV_0 = B, BV_t = BV_{t - 1} - D_t = B - \sum _{k = 1}^t D_k\);
  \textbf{do not include in cash-flow analysis}

  \begin{itemize}
  \tightlist
  \item
    Straight line: \(D_t = \frac{B - S}{N}\)
  \item
    Declining balance:
    \(D_t = BV_{t - 1}d, d = 1 - \left(\frac{S}{B}\right)^\frac{1}{N}\)
    or \(d = \frac{200\%}{N}\) (double declining balance)
  \item
    Sum of the years' digits:
    \(D_t = \frac{N - t + 1}{SOYD}(B - S), SOYD = \frac{N(N + 1)}{2}\)
  \item
    Units of production:
    \(D_t = \frac{\text{Production in year }t}{\text{Total production}}(B - S)\)
  \end{itemize}
\item
  Accounting: each transaction has a debit (increase to left/decrease to
  right) and credit (decrease to left/increase to right)

  \begin{itemize}
  \tightlist
  \item
    Net income = revenue - COGS - SG\&A - depreciation - interest -
    taxes; retained earnings (equity) = net income - dividends
  \item
    Primary accounts: assets (left), liabilities (right), equity
    (right); expenses (left), revenue (right)
  \item
    Label transactions!
  \end{itemize}
\item
  Taxes:
  \(CTF = 1 - \frac{td}{i_A + d}\left(\frac{1 + \frac{i_A}{2}}{1 + i_A}\right), CSF = 1 - \frac{td}{i_A + d}, PW = -FC \cdot CTF + S \cdot CSF \cdot (P/F, i_R, N) + (1 - t) \cdot PW_{\text{Cash Flows}}\)

  \begin{itemize}
  \tightlist
  \item
    \(PW = -FC + A(P/A, i, N) + S(P/F, i, N), AW = PW(A/P, i, N)\)
    (without taxes)
  \end{itemize}
\item
  Inflation: \((1 + i_R)(1 + f) = 1 + i_A\)
\end{itemize}
